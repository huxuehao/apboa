# 技能包管理

技能包是工具的高阶组合形态，将一组相关工具、资源引用、示例和脚本打包为可复用的技能单元，赋予智能体完成特定领域复杂任务的能力。


## 功能概述

- 支持按分类管理技能包
- 技能包可关联多个工具，运行时自动注册到工具集
- 支持资源引用、示例和脚本三种扩展内容
- 支持从 ZIP 压缩包或 Git 仓库导入技能包
- 修改技能包时自动装载脚本到本地


## 核心概念

### 技能包组成

一个技能包由以下部分组成：

| 组成 | 说明 | 格式 |
|------|------|------|
| **基本信息** | 名称、描述、分类 | 文本 |
| **技能内容** | 技能概述，描述该技能的能力和使用方式 | Markdown |
| **资源引用** | 参考文档、知识片段等 | JSON 数组 `[{name, content}]` |
| **示例** | 技能使用的示例说明 | JSON 数组 `[{name, content}]` |
| **脚本** | 可执行的脚本文件 | JSON 数组 `[{name, content}]` |
| **关联工具** | 技能包依赖的工具列表 | 工具 ID 列表 |

### 技能包与工具的关系

- 技能包可以关联 0~N 个工具
- 运行时，技能包关联的工具会自动注册到智能体的工具集中
- 一个工具可以被多个技能包关联

### 导入方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| **ZIP 导入** | 上传 ZIP 压缩包，包含技能定义文件 | 离线分享、批量导入 |
| **Git 导入** | 从 Git 仓库拉取技能定义 | 版本管理、团队协作 |
| **手动创建** | 在线填写技能包信息 | 简单技能、快速创建 |


## 操作指南

### 手动创建技能包

1. 在技能包管理页面点击 **添加新技能包** 卡片
2. 填写表单：
   - **分类**：选择或新建分类
   - **名称**：技能包名称
   - **描述**：技能功能说明
   - **技能内容**：使用 Markdown 编辑器编写技能概述
   - **资源引用**：添加参考文档（名称+内容键值对）
   - **示例**：添加使用示例
   - **脚本**：添加可执行脚本
   - **关联工具**：选择技能包依赖的工具
3. 点击确认提交

### 导入技能包

1. 点击导入按钮
2. 选择导入方式：
   - **ZIP 导入**：上传 ZIP 文件
   - **Git 导入**：填写仓库地址和分支信息
3. 系统自动解析并创建技能包

### 编辑技能包

点击卡片右上角菜单 → **编辑**，修改技能包内容和关联工具。

:::warning
修改技能包后，关联智能体将自动重新注册。脚本变更会自动重新装载到本地。
:::

### 搜索与筛选

- **分类筛选**：页面顶部分段控制器切换分类
- **名称搜索**：输入关键词后回车搜索


## 在智能体中使用

在智能体配置的「工具与能力」步骤中：

1. 在 **技能包** 区域，按分类折叠面板浏览
2. 勾选需要的技能包（Checkbox 多选）
3. 技能包关联的工具会自动注册到智能体的工具集中

## 最佳实践

### 技能内容编写

技能内容是模型理解技能能力的关键，建议包含：

- **技能目标**：该技能解决什么问题
- **使用场景**：什么情况下应该使用该技能
- **操作步骤**：使用该技能的典型步骤
- **注意事项**：使用限制和注意点

### 工具关联建议

- 将逻辑上相关的工具组合到同一技能包
- 避免将不相关工具放入同一技能包
- 确保技能包关联的工具都已启用

### 脚本管理

- 脚本在技能包更新时自动装载到本地文件系统
- 清空脚本内容时，本地脚本文件会自动移除
- 脚本文件路径遵循平台的命名规范
