# 知识库管理

知识库管理是智能体的"私有记忆与专业智库"，通过先进的 RAG（检索增强生成）技术，将企业的私有文档、数据库、业务知识统一构建为可检索、可推理的结构化知识体系，让智能体在通用能力基础上具备企业级专业知识。


## 功能概述

- 支持三种知识库类型：百炼、Dify、RagFlow
- 支持多种 RAG 模式配置
- 提供连接配置、端点配置、检索配置、重排配置等完整参数
- 自动健康检查与同步状态监控
- 支持元数据过滤和查询重写


## 核心概念

### 知识库类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **百炼（BAILIAN）** | 阿里云百炼平台知识库 | 已有百炼平台知识库 |
| **Dify** | Dify 平台知识库 | 已有 Dify 知识库 |
| **RagFlow** | RagFlow 平台知识库 | 已有 RagFlow 知识库 |

### RAG 模式

| 模式 | 说明 |
|------|------|
| **AGENTIC** | 智能体自主决定何时检索知识 |
| **RETRIEVAL** | 每次对话都进行知识检索 |
| **NONE** | 不使用 RAG |

### 配置区域

知识库配置包含以下区域：

| 区域 | 说明 |
|------|------|
| **连接配置** | 知识库服务的连接信息（API Key、URL 等） |
| **端点配置** | 检索和查询的端点地址 |
| **检索配置** | 检索参数（Top-K、相似度阈值等） |
| **重排配置** | 结果重排序参数 |
| **查询重写配置** | 查询改写和优化参数 |
| **元数据过滤** | 基于元数据的条件过滤 |
| **HTTP 配置** | 自定义 HTTP 请求参数 |


## 操作指南

### 创建知识库

1. 在知识库管理页面点击 **添加新知识库** 卡片
2. 选择知识库类型
3. 填写表单：
   - **名称**：知识库名称
   - **描述**：知识库说明
   - **类型**：百炼 / Dify / RagFlow
   - **RAG 模式**：智能体 / 检索 / 无
   - **连接配置**：填写 API Key、服务地址等
   - **端点配置**：检索接口地址
   - **检索配置**：Top-K、相似度阈值等
   - **重排配置**：是否启用重排、重排模型等
   - **查询重写**：是否启用查询重写
   - **元数据过滤**：添加过滤条件
   - **HTTP 配置**：自定义请求头和参数
4. 点击确认提交

### 查看详情

点击卡片右上角菜单 → **查看**，查看完整配置信息，包括：

- 关联的智能体列表
- 连接配置、端点配置、检索配置等
- 健康状态和最后同步时间

### 编辑配置

点击卡片右上角菜单 → **编辑**，修改知识库配置。

:::warning
修改知识库配置后，关联智能体将自动重新注册。
:::

### 搜索与筛选

- **类型筛选**：页面顶部分段控制器切换知识库类型
- **名称搜索**：输入关键词后回车搜索


## 在智能体中使用

在智能体配置的「知识库与 MCP」步骤中：

1. 在 **知识库** 区域，按类型折叠面板浏览
2. 勾选需要关联的知识库（Checkbox 多选）
3. 每个智能体可关联多个不同类型的知识库

## 配置说明

### 百炼知识库连接配置

- **API Key**：百炼平台 API Key
- **知识库 ID**：百炼平台中的知识库标识

### Dify 知识库连接配置

- **API Key**：Dify 平台 API Key
- **Base URL**：Dify 服务地址

### RagFlow 知识库连接配置

- **API Key**：RagFlow 平台 API Key
- **Base URL**：RagFlow 服务地址
- **知识库 ID**：RagFlow 中的知识库标识

### 元数据过滤

通过添加元数据条件，可以在检索时精确过滤文档：

- **字段名**：元数据的字段名
- **比较操作符**：`=`、`!=`、`>`、`<` 等
- **值**：过滤条件的值


## 常见问题

### 知识库健康检查失败？

- 检查 API Key 是否正确
- 检查服务地址是否可达
- 检查网络防火墙设置

### 检索结果不准确？

- 调整 Top-K 参数（增大返回更多结果）
- 调整相似度阈值（降低阈值放宽匹配）
- 启用重排功能提升结果质量
- 添加元数据过滤缩小检索范围
